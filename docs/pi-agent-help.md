# SOPs Module Builder — /help Reference

Complete CLI reference for the Pi-based module builder agent.
See pi-agent-architecture.md for the full system design.

This document is the single reference for:
- All pipeline commands (/new-client, /analyze, /script, /design, /package, /learn)
- Feedback loop commands (/ask-client, /request-ld-review, /request-sme-review)
- Image pipeline commands (/generate-manifest, /generate-images, /review-images, /regenerate, /match-style)
- Audit & reporting commands (/audit, /trace, /coverage, /decisions, /changelog, /timeline)
- Expertise commands (/expertise, /expertise-sync, /expertise-diff)
- Quality gates (hard blocks, soft blocks, warnings)
- Keyboard shortcuts
- Workflow checklists (start of engagement, before delivery, after every module)
- Configuration options
- Worked examples (full client workflow, style-matched module addition, slide tracing, compliance coverage)

The full formatted /help output is maintained in the local working copy.
